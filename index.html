background
background-color

Background-color - определяет цвет фона элемента.

Синтаксис: background-color: цвет;

Универсальная запись: background: цвет;

background-image

Background-image - устанавливает фоновое изображение для элемента либо задает градиентную заливку

Синтаксис: background-image: url("путь к файлу");

Универсальная запись: background: цвет url("путь к файлу");

background-repeat

Background-repeat - определяет, как будет повторяться фоновое изображение. Может иметь следующие значения:

  1. repeat — фоновое изображение повторяется по горизонтали и вертикали;
  2. no-repeat — устанавливает одно фоновое изображение в элементе без его повторений;
  3. repeat-x — фоновый рисунок повторяется только по горизонтали;
  4. repeat-y — фоновый рисунок повторяется только по вертикали;
  5. space — изображение повторяется столько раз, чтобы полностью заполнить область; если это не удаётся, между картинками добавляется пустое пространство;
  6. round — изображение повторяется так, чтобы в области поместилось целое число рисунков; если это не удаётся сделать, то фоновые рисунки масштабируются.

Синтаксис: background-repeat: repeat;

Универсальная запись: background: цвет url("путь к файлу") значение повторения;

background-position

Background-repeat - задает начальное положение фонового изображения, где первое значение (проценты, px, слово) определяет расположение по горизонтали, второе - по вертикали. Может иметь следующие значения:

  1. 0 0 или top left или 0% 0% — значение по умолчанию;
  2. center center или 50% 50% — изображение разместится по центру, если значения повторяются, то можно использовать одну запись(center или 50%);
  3. 0 50% или left — изображение разместится слева по центру;
  4. 100% 100% или bottom right — изображение разместится справа в нижнем углу;
  5. 20px 20px — изображение разместится на 20 пикселей правее и ниже левого верхнего угла или на любую другую заданую величину в пикселях;

Синтаксис: background-position: 0 0 или top left или 0% 0%;

Универсальная запись: background: цвет url("путь к файлу") 0 0 значение повторения;

background-size

Background-size - масштабирует фоновое изображение согласно заданным размерам. В цифрах, где первое значение (проценты, px) определяет ширину, второе - высоту. Либо специальным значением. Может иметь следующие значения:

  1. auto auto — значение по умолчанию;
  2. 100% auto — размеры картинки по ширине блока;
  3. auto 100% — размеры картинки по высоте блока;
  4. 100% 100% — растягиваем картинку на всю ширину и высоту блока;
  5. contain — масштабирует изображение с сохранением пропорций таким образом, чтобы картинка целиком поместилась внутрь блока. То есть по меньшей стороне картинки;
  6. cover — масштабирует изображение с сохранением пропорций так, чтобы его ширина или высота равнялась ширине или высоте блока. То есть по большей стороне картинки;
  7. 200px 150px — устанавливаем конкретные размеры. Если установлено одно значение, оно задает ширину фона, второе значение принимается за auto. Пропорции картинки при этом сохраняются. Использование двух значений через пробел задает ширину и высоту фоновой картинки;

Синтаксис: background-size: 200px auto;

Универсальная запись: background: цвет url("путь к файлу") позиция / cover значение повторения;

background-attachment

Background-attachment - устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента. Может иметь следующие значения:

  1. scroll — позволяет перемещаться фону вместе с окном браузера, значение по умолчанию;
  2. fixed — делает фоновое изображение элемента неподвижным, а так же устанавливает фоновое изображение относительно окна браузера;
  3. local — фон фиксируется с учётом поведения элемента. Если элемент имеет прокрутку, то фон будет прокручиваться вместе с содержимым, но фон выходящий за рамки элемента остаётся на месте.;

Универсальная запись: background: цвет url("путь к файлу") позиция / размер значение повторения fixed;

Установка нескольких фоновых изображений для блока

Свойство background позволяет задать несколько фоновых изображений одному блоку, причем со своими настройками позиционирования масштабирования и прокрутки. Указывать значения следует через запятую.

Универсальная запись: background: цвет url("путь к файлу") позиция / размер значение повторения значение attachment, следующий слой фонового изображения со своимим настройками. Слой, который расположен левее по коду будет накладывать на тот, что правее по коду;
